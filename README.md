    TAKE AWAYS,

        2. Everything in Scala is expressions

        4. Use Recursion when you need Loops
        5. When you need loops user tail recursion,@tailrec
        6. Call By Name, expression is evaluated at every use
        7. Call by Value,value is computed before call and same value is used everywhere
